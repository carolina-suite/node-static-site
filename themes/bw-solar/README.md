
# BW Sandstone #

This is a blog theme based on the bootswatch theme Solar.

## Features #

* Many Social Media Links

## Configuration #

* `socialLinks`: An object containing account names for social media accounts.
  * `facebook`
  * `github`
  * `instagram`
  * `linkedin`
  * `pinterest`
  * `twitter`
  * `vimeo`
  * `youtube`
